CLI / AGENT

命令行与 Agent 接入

CLI Binary

安装后自带 tools\Optimizer233.Agent.exe，支持列出操作、读取风险级别、JSON 输出与自动化执行。

Optimizer233.Agent.exe list --json
Optimizer233.Agent.exe get cleanup-run --json
Optimizer233.Agent.exe exec cleanup-scan --json

Risk-aware Ops

所有 CLI 操作都带 low / medium / high 风险元数据，便于 agent 在执行前做策略判断。
